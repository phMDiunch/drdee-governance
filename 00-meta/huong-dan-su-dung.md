# Hướng dẫn Sử dụng Hệ thống Văn bản

---

## 1. Đọc tài liệu

- Nhân viên đọc trực tiếp trên GitHub: `github.com/phMDiunch/drdee-governance`
- Điều hướng theo thư mục hoặc từ mục lục trong [README.md](../README.md)
- Mỗi file Markdown hiển thị đẹp trực tiếp trên GitHub — không cần tải về

Nếu là lần đầu đọc repo này, mở trước [GOVERNANCE.md](../GOVERNANCE.md).

---

## 2. Hiểu trạng thái văn bản

| Trạng thái        | Ý nghĩa                             |
| ----------------- | ----------------------------------- |
| `Đang áp dụng`    | Văn bản chính thức, có hiệu lực     |
| `Đang soạn thảo`  | Chưa ban hành, không áp dụng        |
| `Đang review`     | Đang trong chu kỳ cập nhật hàng năm |
| `Đã hết hiệu lực` | Chỉ lưu tham khảo, không áp dụng    |

---

## 3. Quy trình cập nhật văn bản

```
Phát sinh nhu cầu thay đổi
        ↓
Người soạn cập nhật file → đổi Trạng thái = "Đang review"
        ↓
BGĐ / người có thẩm quyền xem xét
        ↓
Duyệt → cập nhật Phiên bản, Hiệu lực, Trạng thái = "Đang áp dụng"
        ↓
Thông báo cho các bộ phận liên quan
```

---

## 4. Quy tắc đặt tên file

- Tất cả chữ thường, dấu gạch ngang, không dấu tiếng Việt
- VD: `chinh-sach-luong-bac-si.md`, `quy-trinh-tiep-nhan.md`
- Mỗi vị trí/chủ đề = 1 file riêng — không gộp nhiều nội dung vào 1 file lớn

---

- [GOVERNANCE.md](../GOVERNANCE.md) — để hiểu repo này vận hành thế nào

- [Bản đồ hệ thống văn bản](./ban-do-he-thong-van-ban.md) — để không quên đầu mục quan trọng
- [Template văn bản](./_template-van-ban.md) — để tạo file mới thống nhất
- [Quy ước soạn thảo](./quy-uoc-soan-thao.md) — để giữ cách viết và cập nhật nhất quán

## 6. Ai chịu trách nhiệm gì

| Loại văn bản | Người soạn                 | Người duyệt       |
| ------------ | -------------------------- | ----------------- |
| Quy chế      | BGĐ / CHRO                 | Chủ tịch / HĐQT   |
| Chính sách   | Trưởng phòng liên quan     | Tổng giám đốc     |
| Quy định     | Trưởng phòng               | Giám đốc bộ phận  |
| Quy trình    | Trưởng nhóm / Trưởng phòng | Giám đốc Vận hành |
| Biểu mẫu     | Trưởng nhóm                | Trưởng phòng      |
