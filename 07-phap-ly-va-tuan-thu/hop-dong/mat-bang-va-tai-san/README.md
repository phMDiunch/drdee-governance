# Hợp đồng Mặt bằng và Tài sản

> Thư mục này dành cho các hợp đồng thuê, sử dụng, bảo trì hoặc quản lý tài sản phục vụ hoạt động của công ty.

## Đầu mục dự kiến

- Hợp đồng thuê nhà hoặc mặt bằng
- Phụ lục gia hạn hoặc điều chỉnh giá thuê
- Hợp đồng thuê thiết bị, máy móc
- Hợp đồng bảo trì, bảo dưỡng
- Hợp đồng thuê tài sản vận hành khác
