# Danh mục Trừ Chi phí Cost

> Áp dụng thống nhất cho **toàn bộ** chính sách lương/thưởng của **tất cả bộ phận** tại DR DEE.
> File này là **nguồn duy nhất** định nghĩa chi phí cost bị trừ trước khi tính thưởng. Các file lương bộ phận chỉ **tham chiếu** về đây, không tự định nghĩa lại.

> ⚠️ **Trạng thái**: Bảng chi phí **máng trong (Invisalign)** đã có số đầy đủ. Bảng **vật liệu điều trị** mới có bộ môn Niềng răng (một phần) — các dòng/bộ môn còn lại đang chờ bổ sung. Giữ trạng thái "Đang soạn thảo" đến khi hoàn tất.

---

## 1. Nguyên tắc

Một số dịch vụ có **giá vốn (cost) cao** — tiền công ty chi ra cho vật tư/máng/labo không phải là lợi nhuận. Nếu tính thưởng trên nguyên doanh số/doanh thu sẽ **thưởng vượt quá phần giá trị thực** mà nhân viên tạo ra.

```
Cơ sở tính thưởng = Doanh số / Doanh thu thực − Chi phí cost (theo bảng tương ứng dưới đây)
```

Chi phí cost trừ **theo từng vai trò / loại thưởng**:

- **Thưởng tư vấn** → **chỉ trừ Invisalign / máng trong** (Bảng A, mục 2). **Không** trừ vật liệu mắc cài / implant / răng sứ.
- **Thưởng điều trị** → **trừ tất cả**: Invisalign (Bảng A) **và** vật liệu mắc cài, implant, răng sứ (Bảng B, mục 3).
- **Thưởng Cơ sở** → trừ tổng chi phí cost phát sinh của cơ sở (Bảng A + Bảng B) khỏi doanh thu Cơ sở trước khi tính %.

## 2. Bảng A — Trừ chi phí máng trong (Invisalign) — áp dụng cho **thưởng tư vấn**

| Gói Invisalign     | Mức trừ         |
| ------------------ | --------------- |
| Comprehensive 5    | 45.000.000 đ    |
| Comprehensive 3    | 35.000.000 đ    |
| Moderate           | 35.000.000 đ    |
| First              | 30.000.000 đ    |
| Lite               | 30.000.000 đ    |
| Express            | 20.000.000 đ    |
| Essential          | 15.000.000 đ    |
| Angle Select 30    | 30.000.000 đ    |
| Angle Select 20    | 15.000.000 đ    |
| Angle Select 10    | 10.000.000 đ    |

> Cách dùng: `Thưởng tư vấn = Hệ số thưởng × (Doanh số tư vấn − Mức trừ gói tương ứng)`.

---

## 3. Bảng B — Trừ chi phí vật liệu — áp dụng cho **thưởng điều trị**

> **Lưu ý**: Thưởng điều trị trừ **tất cả** chi phí cost — ngoài Bảng B dưới đây còn trừ **cả chi phí Invisalign (máng trong) theo Bảng A** nếu ca điều trị là Invisalign.

### Bộ môn: Niềng răng

| Dịch vụ                      | Chi phí vật liệu |
| ---------------------------- | ---------------- |
| Mắc cài kim loại thường      | 1.500.000 đ      |
| Mắc cài kim loại tự buộc     | 6.000.000 đ      |
| Mắc cài sứ thường            | 5.000.000 đ      |
| Mắc cài sứ tự buộc           | 10.000.000 đ     |
| Mắc cài mặt lưỡi thường      | 6.000.000 đ      |
| _(còn tiếp — VD: mắc cài mặt lưỡi tự buộc…)_ | _(chờ bổ sung)_ |

### Các bộ môn khác _(chờ bổ sung)_

- Implant _(chờ bổ sung)_
- Răng sứ / phục hình _(chờ bổ sung)_
- Nội nha _(chờ bổ sung)_

> Niềng Invisalign khi tính **thưởng điều trị**: trừ theo chi phí máng trong ở **Bảng A** (đã chốt).

## 4. Áp dụng cho loại thưởng nào

| Loại thưởng | Trừ chi phí cost? | Bảng áp dụng |
| ----------- | ----------------- | ------------ |
| Thưởng % tư vấn | **Có** | **Chỉ Bảng A** (Invisalign / máng trong) |
| Thưởng % điều trị (BS, ĐD) | **Có** | **Bảng A + Bảng B** (Invisalign + mắc cài, implant, răng sứ) |
| Thưởng Cơ sở (quỹ chi nhánh) | **Có** | Tổng cost cơ sở (A + B) |
| Thưởng % giới thiệu — răng sứ, implant (theo % doanh số) | _(chờ anh xác nhận)_ | Đề xuất: Bảng B |
| Thưởng giới thiệu — niềng (mức cố định/khách) | **Không áp dụng** | — (mức cố định, không tính theo %) |

> Nguyên tắc chung: thưởng tính theo **%** của doanh số/doanh thu → **có trừ cost**; thưởng là **mức cố định theo đầu khách/đầu ca** → **không trừ cost**.
