# Khuyến mại theo kỳ (lưu trữ)

> 🔨 **PLACEHOLDER** — chờ cung cấp nội dung.

Nơi lưu **các chương trình khuyến mại/ưu đãi** theo tháng/quý. Mỗi chương trình là 1 file có **ngày bắt đầu – ngày kết thúc**. Giữ lại kể cả khi hết hạn để làm **bằng chứng đối chiếu giá tại thời điểm khách ký hợp đồng**.

## Quy ước đặt tên

- Theo kỳ: `2026-Q3.md`, `2026-07.md` …
- Mỗi file ghi rõ: **hiệu lực từ – đến**, phạm vi áp dụng (Cơ sở/Hệ thống), dịch vụ, mức ưu đãi, điều kiện.

## Lưu ý

- Khuyến mại **không nhúng vào hợp đồng**. Hợp đồng chỉ ghi giá niêm yết; ưu đãi điền theo ca theo chương trình đang hiệu lực khi ký.
