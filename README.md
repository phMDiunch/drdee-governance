## Hệ thống tầng văn bản

```
Tầng 1 — QUY CHẾ       Board ban hành. Nền tảng pháp lý nội bộ.
Tầng 2 — CHÍNH SÁCH    BGĐ ban hành. Review hàng năm.
Tầng 3 — QUY ĐỊNH      Trưởng phòng đề xuất, BGĐ duyệt.
Tầng 4 — QUY TRÌNH     SOP hàng ngày. Trưởng phòng ban hành.
Tầng 5 — BIỂU MẪU      Checklist, form, template thực thi.
```

Văn bản tầng trên có giá trị cao hơn tầng dưới. Khi có mâu thuẫn, tầng trên được ưu tiên.

## Quy ước metadata

Mỗi văn bản phải có header ở đầu file:

```
Loại văn bản : Chính sách / Quy định / Quy trình / Biểu mẫu
Phiên bản    : 1.0
Hiệu lực     : DD/MM/YYYY
Review tiếp  : DD/MM/YYYY
Người ban hành: [Chức danh]
Trạng thái   : Đang áp dụng / Đang soạn thảo / Đã hết hiệu lực
```
